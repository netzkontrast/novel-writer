# Activate Wang Yu Style Creation Mode

Set and activate the Wang Yu style (author of "A Record of a Mortal's Journey to Immortality") creation mode, loading the complete style parameters and world-building system.

## Usage

```
/wangyu-style [options]
```

## Options

- `--intensity <1-10>` - Style intensity (default: 8)
- `--focus <aspect>` - Focus on enhancing a certain aspect:
  - `world` - World-building
  - `character` - Character creation
  - `suspense` - Suspense setting
  - `resource` - Resource system
  - `battle` - Battle description

## Function Description

After activation, it will automatically:

1. **Load the Style Template**
   - A calm and objective narrative style.
   - A rigorous cultivation world-building.
   - A complex network of character relationships.

2. **Initialize the World Setting**
   - An eight-level cultivation realm system.
   - A four-category cultivation resource system.
   - A diverse power structure of factions.

3. **Configure Creation Parameters**
   - Narrative distance: Detached and objective.
   - Emotional intensity: Restrained and introverted.
   - Description density: Detailed and precise.
   - Suspense level: Layered and progressive.

4. **Enable Special Features**
   - Foreshadowing tracking system.
   - Resource scarcity check.
   - Power system validation.
   - Character behavior logic review.

## Notes

- This mode is particularly suitable for Xianxia and Xuanhuan genres.
- It is recommended to use it in conjunction with the `/track` command to manage foreshadowing.
- It can be combined with other writing methods (such as the Hero's Journey).
- After activation, it will affect all subsequent creation commands.

## Related Commands

- `/wangyu-write` - Write in the Wang Yu style.
- `/wangyu-analyze` - Analyze the style conformity.
- `/wangyu-enhance` - Enhance the Wang Yu style features.
- `/expert wangyu-master` - Summon the Wang Yu style master.
